
# 📚 Curso de Machine Learning – Notebooks y Recursos

Este repositorio contiene todos los notebooks, datasets y materiales utilizados en el curso de **Machine Learning**, orientado a proporcionar una formación práctica y aplicada en técnicas modernas de aprendizaje automático.

## 🎯 Objetivo del repositorio
Centralizar y compartir de forma organizada los recursos necesarios para que los estudiantes puedan:
- Comprender los fundamentos del Machine Learning.
- Aplicar técnicas de regresión y clasificación.
- Entrenar modelos utilizando buenas prácticas.
- Experimentar con diferentes algoritmos y herramientas.
- Ejecutar los notebooks directamente desde Google Colab sin necesidad de instalación local.
